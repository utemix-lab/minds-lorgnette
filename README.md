# Mind's Lorgnette

Среда для разработки производных от extended-mind широкого назначения.

Project Status: Concept  
License: Apache 2.0

---

## Назначение

**Mind's Lorgnette** — исследовательская среда для интеграции различных способов организации информации и знаний.

Проект изучает, как разные формы представления мышления могут сосуществовать в одной системе и использоваться совместно:

- таблицы и структурированные данные,
- графы и сети связей,
- майнд-карты и иерархии,
- канбаны и состояния,
- диаграммы и визуальные маршруты,
- текстовые и нарративные формы.

Цель — разработка универсального приложения для работы со знаниями **на этапе их формирования**, а не только хранения.

## Роль в экосистеме utemix-lab

Mind's Lorgnette является производной от **extended-mind** и предназначена для более широкого, внеэкосистемного применения.

| Проект              | Роль |
|---------------------|------|
| extended-mind       | Мета-координатор экосистемы, архитектурные принципы |
| minds-lorgnette     | Универсальная среда работы со знаниями |
| dream-graph         | 3D-визуализация графов |
| vovaipetrova-site   | Классический web-интерфейс |
| vovaipetrova-core   | Контент и перформанс |
| evoquant            | Исследовательские проекты |

## Что делает minds-lorgnette

- Исследует способы организации знаний и мышления
- Разрабатывает универсальное приложение
- Интегрируется с существующими структураторами знаний
- Расширяет принципы extended-mind за пределы экосистемы
- Поддерживает работу с неопределёнными, противоречивыми и незрелыми знаниями

### 3. Семантические пайплайны

Mind's Lorgnette развивает концепцию **конфигурируемых семантических стеков**:

```yaml
# mind-stack.yaml
version: "0.1"
stack:
  ontology:
    driver: yaml_hierarchy
    config: ./categories.yaml
  semantics:
    driver: transformer_embeddings
    model: intfloat/multilingual-e5-small
  connectivity:
    driver: graph_knowledge_base
    backend: neo4j
```

Структура проекта:

```
minds-lorgnette/
├── docs/
│   ├── 0_context/      # Видение и контекст
│   ├── 1_core/         # Концепции ядра
│   ├── philosophy/     # Философские контракты и рамки
│   └── 3_project/      # Дорожная карта
├── mindflow/           # Журнал исследований
├── ecosystem/          # Экосистемный слой
└── prototype/          # Код прототипа (в разработке)
```

Для кого это
Исследователи ИИ — эксперименты с RAG, графами, агентами

Команды — адаптивные базы знаний для сложных проектов

Индивидуальные пользователи — персональные среды мышления
