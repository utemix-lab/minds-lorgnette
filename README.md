# Mind's Lorgnette

**Среда для разработки производных от extended-mind широкого назначения.**

[![Project Status: Concept](https://img.shields.io/badge/Status-Concept-blueviolet)](https://github.com/utemix-lab/minds-lorgnette)
[![License: Apache 2.0](https://img.shields.io/badge/License-Apache%202.0-blue.svg)](LICENSE)

---

## Назначение

Mind's Lorgnette — среда для исследования **интеграции различных способов организации информации и знаний**:

- Таблицы
- Графики
- Графы
- Майнд-карты
- Канбаны
- Диаграммы связей

А также интеграция с культовыми структураторами:
- **Obsidian**
- **Notion**
- **Roam Research**
- и другие

Цель — разработка **универсального приложения** для работы со знаниями.

---

## Роль в экосистеме utemix-lab

Mind's Lorgnette — производная от extended-mind для более широкого, не внутри-экосистемного назначения.

| Проект | Роль |
|--------|------|
| **extended-mind** | Мета-координатор экосистемы |
| **minds-lorgnette** | Универсальное приложение для работы со знаниями |
| **dream-graph** | 3D визуализация графов |
| **vovaipetrova-site** | Классический web-интерфейс |
| **evoquant** | Исследовательские проекты |

### Что делает minds-lorgnette

- **Исследует** интеграцию способов организации знаний
- **Разрабатывает** универсальное приложение
- **Интегрирует** с Obsidian, Notion и другими структураторами
- **Расширяет** возможности extended-mind за пределы экосистемы

### Связь с экосистемой

```
extended-mind (ядро)
        ↓ принципы, архитектура
minds-lorgnette (производная)
        ↓ интеграция с внешними инструментами
Универсальное приложение
```

## Направления исследований

### 1. Способы организации информации

| Способ | Описание | Статус |
|--------|----------|--------|
| Таблицы | Структурированные данные | Исследование |
| Графы | Связи между понятиями | Исследование |
| Майнд-карты | Иерархии и ассоциации | Планируется |
| Канбаны | Потоки и состояния | Планируется |
| Графики | Визуализация данных | Планируется |

### 2. Интеграция с инструментами

| Инструмент | Интеграция | Статус |
|------------|------------|--------|
| Obsidian | Markdown, графы, плагины | Исследование |
| Notion | Базы данных, API | Планируется |
| Roam Research | Блочная структура | Планируется |
| Logseq | Outliner, графы | Планируется |

### 3. Семантические пайплайны

Mind's Lorgnette развивает концепцию **конфигурируемых семантических стеков**:

```yaml
# mind-stack.yaml
version: "0.1"
stack:
  ontology:
    driver: yaml_hierarchy
    config: ./categories.yaml
  semantics:
    driver: transformer_embeddings
    model: intfloat/multilingual-e5-small
  connectivity:
    driver: graph_knowledge_base
    backend: neo4j
```

## Для кого это?

- **Исследователи ИИ** — гибкая среда для экспериментов с RAG, графами, агентами
- **Разработчики** — создание нишевых инструментов для работы с информацией
- **Команды** — адаптивная база знаний для сложных проектов
- **Индивидуальные пользователи** — универсальное приложение для личных знаний
